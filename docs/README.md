## 3. 维护规则

- 当前实现发生变化时, 优先回写正式事实源文档。
- TODO 文档只描述“在当前实现基础上还要做什么”, 不保留已经失效的前置方案。
- 迁移完成后, 规划文档应明确标记为“归档”或“历史”, 不继续伪装成未实施方案。
- 若 `src/http_client/webview/ui/*` 等过渡目录仍然存在, 文档必须明确其是否为主运行路径, 不能仅按目录存在与否下结论。
